远程控制台

这是用浏览器控制这台 Windows 主机的入口。普通使用只需要按下面三步走；分辨率、帧率、码率这些高级选项已经放在右侧画质区，默认保持自动即可。

1 连接主机 首次连接输入 PIN；通过后浏览器会保存令牌，下次可自动恢复。
2 启动画面 先打开视频预览，确认看到桌面后再进行键鼠控制。
3 启用键鼠 需要操作远端时再开启；点进画面后键盘和鼠标才会发送到主机。
